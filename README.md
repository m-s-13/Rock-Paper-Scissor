# Rock-Paper-Scissor
Welcome to the GAME: Rock-Paper-Scissor

<h3>Rules of the GAME</h3>
                   
                   +Rock vs Paper --> Paper wins
                   +Paper vs Scissor --> Scissor wins
                   +Scissor vs Rock --> Rock wins
                   
                   +Enter numbers 1,2,3 to avoid error message.
                   
                   +Using keyboard type:
                                        1 for *Rock
                                        2 for *Paper
                                        3 for *Scissor
